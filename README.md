# Student Support Hub

A simple web platform designed to help students manage stress and support each other.

## Features
- 60-second stress reset breathing exercise
- Daily mood check-in
- Anonymous positivity wall
- Local storage for privacy

## Tech Stack
- HTML
- CSS
- Tailwind CSS
- JavaScript

## How to Run
1. Download the repository
2. Open index.html in a browser
